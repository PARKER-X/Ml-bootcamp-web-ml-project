Confusion matrix

In this tutorial we will discuss about confusion matrix

A Confusion Matrix is a table that is often used to describe the performance of a classificatiob model(or classifier) on a set of tets data for which the true values are known

Let's now define the most basic terms : We take a simple example of disease predictor

  • true positives (TP): These are cases in which we predicted yes (they have the disease), and they do have the disease.
  • negatives (TN): We predicted no, and they don't have the disease.
  • false positives (FP): We predicted yes, but they don't actually have the disease.
  • false negatives (FN): We predicted no, but they actually do have the disease.
