Mnist

  • Set of 70k small images of digits handwritten by high school students and employess of the us census bureau
  • All images are labelled with the respective digits they represent
  • Mnist is the hello world of machine learning
  • There are 70k images and each images has 28*28 features[784]
  • Each features simply represnts 1 pixel-intensity from 0 to 255. if the intensity is 0, it means that the pixel is white and if it's 255 it means it's black

Import modules

 
        
import numpy as np
import pandas as pd
from sklearn.datasets import load_digits

load the data (This data is already install in sklearn) 

    
mnist = load_digits()
mnist.data.shape      #to check shape of data
output:  (1797, 64)

from sklearn.model_selection import train_test_split
x_train, x_test, y_train, y_test = train_test_split(mnist.data,mnist.target,train_size=0.8,random_state=0)


Now check shape of xtrain and xtest

x_train.shape, x_test.shape
output:   ((1437, 64), (360, 64))


Train the model using logistic regression

from sklearn.linear_model import LogisticRegression   #Don't worry in future we learned about logistic regression
clf = LogisticRegression(tol=0.1)
clf.fit(x_train, y_train)


Now, predict your model for top 10 values 

clf.predict(x_test[0:10])
output:  array([2, 8, 2, 6, 6, 7, 1, 9, 8, 5])
